# Theme

Theme can be configured by passing theme object to theme component. It is required to one `Theme` instance be provided as parent to
all components from this library.

```tsx copy
<Theme theme={theme}>
  <App />
<Theme>

```

## Customisation

```tsx
export type ThemeProps = {
  helpers: ThemeHelpers
  size: ElementSize
  variant: ElementVariant
  theme: ElementTheme
  zIndex: ThemeZIndex
  styles: ThemeStyles
  texts: ThemeTexts
  respectReducedMotion: boolean
  dir: ThemeDisplayDirection
}
```
