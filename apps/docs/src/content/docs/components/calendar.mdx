### Dynamic value type

Type of value and onChange callback depends on the selected calendar mode.

```ts copy
type CalendarDateValue = Date | null | undefined
```

| Mode  | Value type                               |
| ----- | ---------------------------------------- |
| date  | `CalendarDateValue`                      |
| range | `[CalendarDateValue, CalendarDateValue]` |
